# 🧩 ivarjt's Dotfiles

My personal dotfiles for Arch Linux with Hyprland.

## 🖥️ Environment

- **OS**: Arch Linux  
- **WM**: Hyprland  
- **Shell**: Zsh  
- **Terminal**: Kitty  
- **Status Bars**: Waybar  

## 📦 Setup

Clone the repo:

**SSH:**
```bash
git clone git@github.com:ivarjt/dotfiles.git ~/dotfiles
```

**HTTPS:**
```bash
git clone https://github.com/ivarjt/dotfiles.git ~/dotfiles
```
